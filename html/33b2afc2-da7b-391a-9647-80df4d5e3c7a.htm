Remarks
While not obvious from the R documentation, it seems that command line arguments need to be passed to get the startup parameters taken into account. Passing the StartupParameter to the API seems not to work as expected. While this function may appear like an oddity to a reader, it proved necessary to the initialisation of the R engine after much trial and error.
